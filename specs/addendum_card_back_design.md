# 追加要望: カード裏面デザインの差別化

## 概要
部隊カード（Troop）と戦術カード（Tactic）の裏面デザインを区別し、プレイヤーが視覚的に識別できるようにする。

## 変更点

### UI (`src/board/Card.tsx`)
*   **`faceDown` 時のレンダリング**:
    *   `card.type` に基づいて条件分岐する。
*   **デザイン**:
    *   **共通**:
        *   Tailwind CSS を使用したスタイリング。
        *   中央にテキストまたはアイコンを表示して分かりやすくする。
    *   **Troop (部隊カード)**:
        *   背景色: `bg-slate-700` (または既存の色)
        *   ボーダー: `border-slate-500`
        *   テキスト: "TROOP" (色は `text-slate-400`)
    *   **Tactic (戦術カード)**:
        *   背景色: `bg-amber-900/80` (戦術的な雰囲気を出すため茶色系)
        *   ボーダー: `border-amber-700`
        *   テキスト: "TACTIC" (色は `text-amber-200`)

## 確認事項
*   相手の手札（裏向き）が、それぞれのカードタイプに応じたデザインで表示されること。
*   デッキ（裏向き）が、それぞれのタイプに応じたデザインで表示されること。
