# 追加要望: 盤面から手札へのカード回収

## 概要
ユーザーが盤面（フラッグスロット、戦術ゾーン）に配置したカードを、自分の手札に戻せるようにする。

## 変更点

### 1. ロジック (`src/moves.js`)
*   `moveCard` のバリデーション変更:
    *   `to.area === 'hand'` への移動を許可する。
    *   **制限事項**:
        *   移動元 (`from`) は `board` でなければならない（デッキや捨て札からの回収は不可）。
        *   移動先 (`to`) のプレイヤーIDは、操作プレイヤー (`ctx.currentPlayer`) と一致しなければならない。

### 2. UI (`src/board/Board.tsx`)
*   **手札エリアのクリック対応**:
    *   手札エリア（カードがない部分も含む）をクリックした際、`activeCard` が盤面のカードであれば「手札に戻す」アクションを実行する。
    *   実装方法:
        *   `Hand` コンポーネントをラップする `div` または `Hand` コンポーネント自体に `onClick` (または `onZoneClick` 的なもの) を追加する。
        *   `Board.tsx` でそのハンドラを受け取り、`moves.moveCard({ ..., to: { area: 'hand' } })` を実行する。

## テスト
*   盤面のカードを手札に戻せること。
*   相手の盤面のカードは戻せないこと（既存チェックでカバー）。
*   捨て札やデッキからは戻せないこと。
