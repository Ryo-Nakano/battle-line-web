# モバイル フローティングカード表示機能

## 概要

モバイルUIで手札からカードを選択した後、モーダルを閉じても「現在選択中のカード」がわかるように、画面右下にフローティングカードを表示する。

## 要件

| 項目 | 仕様 |
|------|------|
| 表示条件 | 手札モーダルが閉じている & カードが選択されている |
| 表示位置 | 画面右下 (ENDボタンの上) |
| サイズ | 通常カードの約70% |
| タップ時動作 | 手札モーダルを再度開く |

## 対象ファイル

### [MODIFY] MobileHand.tsx

`src/board/MobileHand.tsx`

#### 変更内容

1. **FloatingCard コンポーネントの追加**
   - 選択中のカードを70%サイズで表示
   - `position: fixed` で画面右下に配置
   - ENDボタンと重ならない位置調整
   - タップで手札モーダルを開く

2. **MobileHand に FloatingCard を追加**
   - 条件: `!isExpanded && activeCard` (モーダル閉じ & カード選択中)
   - propsで選択中のカードと開くハンドラを渡す

### [MODIFY] MobileBoard.tsx

`src/board/MobileBoard.tsx`

#### 変更内容

1. **MobileHand に `activeCard` オブジェクトを渡す**
   - 現在は `activeCardId` のみ渡しているが、フローティングカード表示のためにカード情報全体が必要

## UI設計

```
┌──────────────────────────────┐
│  MobileGameInfo              │
├──────────────────────────────┤
│                              │
│      戦場エリア               │
│                              │
├──────────────────────────────┤
│  [HAND][3][5][7]    ┌───┐   │
│                     │ 5 │ ← フローティングカード (70%)
│                     └───┘   │
│            [ACTIVATE] [END] │
└──────────────────────────────┘
```

## 検証方法

1. 手札プレビューをタップしてモーダルを開く
2. カードを選択 → モーダルが閉じる
3. 画面右下にフローティングカードが表示されることを確認
4. フローティングカードをタップ → モーダルが再度開くことを確認
5. カードを選択解除 → フローティングカードが消えることを確認
