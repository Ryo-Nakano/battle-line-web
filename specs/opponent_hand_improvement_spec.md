# 相手の手札表示の改善 (Opponent Hand Display Improvement)

## 概要
現在、相手の手札は小さな矩形 (`div`) のリストとして表示されているため、視認性が低く、またカードの種類（部隊/戦術）が判別できない。
これを `Card` コンポーネントを使用した表示に変更し、カードタイプに応じた裏面デザインを表示する。

## 変更内容

### `src/board/Board.tsx`

1.  **コンポーネントの置換**:
    *   相手の手札を表示している `map` ループ内の `div` を、`Card` コンポーネントに置き換える。
2.  **プロパティ設定**:
    *   `card`: 元のカードデータを使用するが、**強制的に `faceDown: true` を適用する**。
        *   例: `card={{ ...card, faceDown: true }}`
    *   `isInteractable`: `false` (操作不可)。
    *   `className`: サイズ調整のため、必要に応じて `scale` や `w`/`h` クラスを調整する。ただし、`Card` コンポーネントのデフォルトサイズ（`w-16 h-24` 等）を活かしつつ、ヘッダーに収まるようにする。
3.  **レイアウト調整**:
    *   コンテナの `flex -space-x-2` を調整し、カードの重なり具合を適切にする（カードが大きくなるため、重なり幅も大きくする必要があるかもしれない。例: `-space-x-12` など、あるいは `scale-75` 程度に縮小して `-space-x-4`）。

## 期待される挙動
*   相手の手札が、単なる四角形ではなく、カードらしいデザイン（裏面）で表示される。
*   部隊カードは灰色（Zinc）ベースの裏面、戦術カードは茶色（Amber）ベースの裏面で表示され、種類が区別できる。
*   手札の枚数が正しく反映される。
